# 0.0.1-dev-22 — postfix `++` / `--`

_Released 2026-05-26_

Postfix increment / decrement. Pairs naturally with the
compound assignment operators that landed in dev-21 and
cuts another classic `i = i + 1` from idiomatic loops.

## Highlights

```mint
import { println } from "mint:io";

export function main(): void {
  let acc: int = 0;
  for (let i: int = 0; i < 5; i++) {
    acc += i;
  }
  if (acc == 10) { println("sum=10"); }

  let n: int = 3;
  n--;
  n--;
  if (n == 1) { println("dec ok"); }
}
```

Rules:

- Operand must be a numeric type (`int`, `int64`, `float`,
  `double`) — `DLM4078` otherwise.
- Operand must be a mutable assignment target (same rule
  the regular `=` already enforces).
- Threaded into the `call()` postfix chain, so
  `arr[i]++` and `obj.field++` Just Work.

Lowered to the matching C++ postfix operator (`(i++)`),
so behavior matches C++ — the value of the expression is
the *pre*-increment value.
