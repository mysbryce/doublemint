# 0.0.1-dev-15 — `doublemint fmt`

_Released 2026-05-26_

Light-touch formatter so `.dlm` files stay tidy without
sacrificing comments or unusual constructs. This release
focuses on the boring-but-load-bearing whitespace passes;
deeper AST-driven reformatting can come later once the
language settles.

## Highlights

```
$ doublemint fmt ugly.dlm        # prints formatted output to stdout
$ doublemint fmt ugly.dlm --write  # rewrites in place
wrote /work/ugly.dlm
$ doublemint fmt ugly.dlm --check  # exit code 1 if changes needed
fmt: /work/ugly.dlm needs formatting
```

What the pass does:

- Converts `\r\n` (and stray `\r`) to `\n`.
- Strips trailing spaces / tabs on every line.
- Collapses runs of two or more blank lines down to a
  single blank.
- Drops leading blank lines from the top of the file.
- Ensures the file ends with exactly one newline.

`--write` and `--check` are mutually exclusive; without
either flag the formatted text goes to stdout so you can
diff it or pipe it elsewhere.

## Files touched

- `src/format/formatter.ts` — new `formatSource(source)`

- `tests/unit/formatter.test.ts` — covers the CRLF,
