# 0.0.1-dev-5 — `mint:math` complete + `mint:base64`

_Released 2026-05-26_

`mint:math` graduates from a four-function teaser into a real
math library, and `mint:base64` lands so Mint programs can talk
to APIs without reaching for an external helper.

## Highlights

### `mint:math` — full surface

| Category | Functions / constants |
| --- | --- |
| Constants | `PI`, `E`, `INFINITY` |
| Trigonometry | `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `atan2` |
| Powers / logs | `sqrt`, `cbrt`, `pow`, `exp`, `log`, `log2`, `log10` |
| Rounding | `floor`, `ceil`, `round`, `trunc`, `floorToInt`, `ceilToInt`, `truncToInt`, `roundToInt` |
| Absolute / sign | `absInt`, `absFloat`, `signInt`, `signFloat` |
| Comparison | `minInt`, `maxInt`, `minFloat`, `maxFloat`, `clampInt`, `clampFloat` |
| Bridging | `intToFloat` |

Most of these route straight through `<cmath>` (`std::sin`,
`std::log`, …) so there is no per-call overhead vs. native
C++. The `*ToInt` and `sign*` helpers live in
`src/runtime/sources/math.cpp` because they need a
`static_cast<int>(std::floor(x))` wrapper.

### `mint:base64`

Single `Base64` namespace, four functions:

```mint
import { Base64 } from "mint:base64";

Base64.encode("Hello, world!");         // "SGVsbG8sIHdvcmxkIQ=="
Base64.decode("SGVsbG8sIHdvcmxkIQ=="); // "Hello, world!"
Base64.encodeBytes([72, 105]);          // "SGk="
Base64.decodeBytes("SGk=");             // [72, 105]
```

The encoder pads with `=` to a 4-byte boundary; the decoder
ignores `=`, whitespace, and any non-alphabet byte (so it
tolerates wrapped input).

## Files touched

- `src/builtins/mintModules.ts` — extends `Math` with 25 entries
  (constants + functions) and registers a fresh
  `mint:base64` module exposing a `Base64` namespace.
- `src/runtime/sources/math.cpp` — `floorToInt` / `ceilToInt` /
  `truncToInt` / `intToFloat` / `signInt` / `signFloat`
  wrappers around the std:: primitives.
- `src/runtime/sources/base64.cpp` — alphabet table + encode /
  decode for strings and byte arrays.
- `src/emitter/cppEmitter.ts` — `"mint:math": "math"` and
  `"mint:base64": "base64"` entries in the source-module map.
- `tests/integration/mintMathBase64Build.test.ts` — covers
  trig / log / floor / ceil / abs / clamp / min / max / sign
  on the math side, plus base64 string + byte round-trips.
