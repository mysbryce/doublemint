# 0.0.1-dev-23 — prefix `++` / `--`

_Released 2026-05-26_

Symmetry with dev-22: prefix increment / decrement. Same
type and mutability rules as the postfix form, but the
expression value reflects the *post*-update value (matches
C++ semantics).

## Highlights

```mint
import { println } from "mint:io";

export function main(): void {
  let n: int = 5;
  ++n;
  ++n;
  if (n == 7) { println("inc"); }

  let m: int = 5;
  --m;
  --m;
  if (m == 3) { println("dec"); }

  let x: int = 0;
  let y: int = ++x;
  if (y == 1) { println("prefix returns new"); }
}
```

Rules:

- Numeric operand required (`DLM4078`, shared with
  postfix).
- Mutable assignment target required (same rule as `=` /
  postfix).
- Result is the value after the update — distinct from the
  postfix form which returns the pre-update value.

Lowered to `(++(arg))` / `(--(arg))` — same expression
shape as the existing unary `-` / `!`, just with the
post-update semantics that C++ provides natively.
