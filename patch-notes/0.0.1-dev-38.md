# 0.0.1-dev-38 — `mint:fmt`

_Released 2026-05-27_

A small formatting module — padding, repeat, hex / oct /
bin radix conversion, fixed-precision floats, and
thousands-separated integers.

## Highlights

```mint
import { println } from "mint:io";
import { Fmt } from "mint:fmt";

export function main(): void {
  println(Fmt.padLeft("42", 6, "0"));         // 000042
  println(Fmt.padRight("mint", 8, "."));      // mint....
  println(Fmt.repeat("=", 10));               // ==========
  println(Fmt.hex(255));                      // ff
  println(Fmt.hexUpper(255));                 // FF
  println(Fmt.octal(8));                      // 10
  println(Fmt.binary(10));                    // 1010
  println(Fmt.precision(3.14159, 2));         // 3.14
  println(Fmt.withThousands(1234567, ","));   // 1,234,567
}
```

## API

| Symbol | Signature |
| --- | --- |
| `Fmt.padLeft` | `(text: string, width: int, pad: string): string` |
| `Fmt.padRight` | `(text: string, width: int, pad: string): string` |
| `Fmt.repeat` | `(text: string, count: int): string` |
| `Fmt.hex` | `(value: int64): string` |
| `Fmt.hexUpper` | `(value: int64): string` |
| `Fmt.octal` | `(value: int64): string` |
| `Fmt.binary` | `(value: int64): string` |
| `Fmt.precision` | `(value: double, places: int): string` |
| `Fmt.withThousands` | `(value: int64, separator: string): string` |

## Files touched

- `src/runtime/sources/fmt.cpp` — nine helpers backed by
  `std::ostringstream`, `std::setprecision`, manual digit
  walks for binary / thousands grouping.
- `src/builtins/mintModules.ts` — registers `mint:fmt`
  with `<iomanip>` / `<sstream>` includes and every helper
  signature.
- `src/emitter/cppEmitter.ts` — `"mint:fmt": "fmt"` entry
  in `RUNTIME_SOURCE_MODULES` so the emitter inlines the
  runtime translation unit when the program imports it.
- `tests/integration/mintFmtBuild.test.ts` — exercises
  every helper end-to-end.
