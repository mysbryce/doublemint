# 0.0.1-dev-27 — modulo `%`

_Released 2026-05-26_

Plain integer remainder joins the arithmetic operator
family. Behaves like C++ `%` — works on `int` and `int64`
(plus untyped numeric literals), rejected on floats with
`DLM4080` so the user doesn't accidentally hit the
"operator `%` is not defined for `double`" C++ surprise.

## Highlights

```mint
import { println } from "mint:io";

export function main(): void {
  if (10 % 3 == 1) { println("ok 10%3"); }
  if (15 % 5 == 0) { println("ok 15%5"); }

  for (let i: int = 0; i < 6; i++) {
    if (i % 2 == 0) {
      println("even " + i.toString());
    }
  }
}
```
