# 0.0.1-dev-13 — `doublemint info`

_Released 2026-05-26_

A small CLI surface for discoverability — `doublemint info`
walks the builtin manifest and prints every `mint:*` module
together with the kind of each thing it exports.

## Highlights

```
$ doublemint info
doublemint 0.0.1-dev-13

Builtin modules:
  mint:array  ->  Array (namespace)
  mint:async  ->  Async (namespace)
  mint:base64  ->  Base64 (namespace)
  mint:collections  ->  Queue (class), Set (class), Stack (class)
  mint:crypto  ->  Crypto (namespace)
  mint:db  ->  KV (namespace)
  mint:fs  ->  File (namespace), Path (namespace)
  mint:http  ->  HeaderMap (class), HttpResponse (class), Fetch (namespace), WebSocket (class), Context (class), Http (class)
  mint:io  ->  IO (namespace), print (function), println (function)
  mint:json  ->  Json (namespace)
  mint:log  ->  Log (namespace)
  mint:math  ->  Math (namespace)
  mint:memory  ->  Memory (namespace)
  mint:net  ->  Url (namespace), Http (namespace)
  mint:os  ->  OS (namespace), Env (namespace)
  mint:process  ->  Process (namespace)
  mint:regex  ->  Regex (class)
  mint:schema  ->  ValidationResult (class), Schema (class)
  mint:simd  ->  Simd (namespace)
  mint:sql  ->  SqlResult (class), Database (class)
  mint:string  ->  String (namespace)
  mint:term  ->  Terminal (namespace)
  mint:test  ->  Test (namespace)

22 modules total.
```

The command exits successfully and is safe to run anywhere
(no project is required). Useful both for newcomers
exploring the stdlib surface and for build scripts that need
to know which modules are present in a given toolchain.
