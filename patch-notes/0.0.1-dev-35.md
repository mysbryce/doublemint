# 0.0.1-dev-35 — `for-of`

_Released 2026-05-27_

Range-based iteration. `for (let x of array) { ... }` lowers
to `for (const auto& x : array)` in C++.

## Highlights

```mint
let xs: int[] = [1, 2, 3, 4, 5];
let sum: int = 0;
for (let x of xs) {
  sum += x;
}

for (let name: string of ["alpha", "beta", "gamma"]) {
  println(name);
}
```

Rules:

- `of` is a contextual keyword; parser uses a small
  lookahead so existing identifiers named `of` keep
  working in other positions.
- Binding kind: `let` (mutable) or `const` (immutable);
  optional explicit type annotation checked against the
  element type.
- Iterable must be an array (`DLM4084` otherwise).
