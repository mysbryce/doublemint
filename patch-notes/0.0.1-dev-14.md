# 0.0.1-dev-14 — `doublemint init` scaffold

_Released 2026-05-26_

One-command project bootstrap. `doublemint init <dir>`
creates the target directory if it doesn't exist and drops a
`main.dlm` hello-world template (using the project directory
name as the embedded greeting).

## Highlights

```
$ doublemint init my-app
wrote /work/my-app/main.dlm
next: doublemint build /work/my-app/main.dlm --out /work/my-app/my-app.exe

$ cat my-app/main.dlm
import { println } from "mint:io";

export function main(): void {
  let name: string = "my-app";
  println("hello from ${name}");
}

$ doublemint build my-app/main.dlm --out my-app/app.exe
$ ./my-app/app.exe
hello from my-app
```

- Without an argument, scaffolds into the current directory.
- Refuses to overwrite an existing `main.dlm` so you can't
  accidentally lose work.
- Prints the suggested next command (with the platform-correct
  `.exe` suffix) so first-time users have a copy-pasteable
  next step.
