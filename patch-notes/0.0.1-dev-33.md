# 0.0.1-dev-33 — hex / oct / bin literals + numeric separator

_Released 2026-05-27_

Modern numeric literals: `0xFF`, `0o755`, `0b1010`, plus the
underscore digit separator `1_000_000`.

## Highlights

```mint
let hex: int = 0xFF;        // 255
let oct: int = 0o755;       // 493
let bin: int = 0b1010;      // 10
let big: int64 = 1_000_000; // 1000000
let pi: double = 3.141_592;
```

Notes:

- `0x` is case-insensitive on prefix + hex digits.
- `0o` is Mint-only; the emitter rewrites it to the C++
  octal `0` prefix.
- `0b` survives into the emitted C++ (C++14+ supports it).
- Underscores are stripped before parsing and before
  emission.
