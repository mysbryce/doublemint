# 0.0.1-dev-29 — compound `%= &= |= ^=`

_Released 2026-05-27_

Round out the assignment family: the integer operators
that landed in dev-27 (`%`) and dev-28 (`& | ^`) all get
their compound-assignment shortcuts.

## Highlights

```mint
import { println } from "mint:io";

export function main(): void {
  let n: int = 12;
  n %= 5;
  if (n == 2) { println("mod-eq ok"); }

  let mask: int = 255;
  mask &= 15;
  if (mask == 15) { println("and-eq ok"); }

  let bits: int = 1;
  bits |= 4;
  bits |= 16;
  if (bits == 21) { println("or-eq ok"); }

  let xv: int = 240;
  xv ^= 15;
  if (xv == 255) { println("xor-eq ok"); }
}
```

Rules:

- LHS must be a mutable assignment target (same rule as
  `=`, shared with the dev-21 compound family).
- Both operands must be integer (`int` / `int64` / numeric
  literal); `DLM4082` otherwise.
- Result type is the LHS type, so the operator is an
  expression (`while ((mask &= 0x0F) != 0)` works).
